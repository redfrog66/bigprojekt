# SZFM - Nagy projekt

## Projekt leírás

Egy időjárás jelentésre alkalmas webapplikáció, melyhez Pythont, Django-t és TensowFlow-ot fogunk használni alapul. A Projekt beadandó a Szoftverfejlesztési Módszertanok tárgyunkra.
